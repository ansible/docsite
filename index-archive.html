Notice on domain name changes

We're moving the docs.ansible.com subdomain to Read The Docs hosting. At the end of the migration effort, you will be able to find documentation for all Ansible community ecosystem projects under the docs.ansible.com subdomain. However the legacy documentation for Ansible Automation Controller will no longer be available from docs.ansible.com. We will use redirects to avoid broken links but Ansible Automation Controller documentation will be hosted from the legacy-controller-docs.ansible.com subdomain only. You can read more about the migration in forum post.
